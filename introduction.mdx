## Core Features

The platform includes several key features designed to enhance your documentation workflow. It supports real-time collaboration, version control, and seamless integration with popular development tools. You can create, edit, and manage documentation pages using a simple markdown-based interface.

<Tabs>
  <Tab title="Web Interface" icon="monitor">
    Access your documentation through an intuitive web dashboard with drag-and-drop functionality.
    
    <Steps>
      <Step title="Sign Up" icon="user-plus">
        Create an account by visiting the registration page and providing your email and password.
      </Step>
      <Step title="Verify Email" icon="mail">
        Check your inbox for a verification link and click to activate your account.
      </Step>
      <Step title="Create Project" icon="folder-plus">
        Navigate to the projects section and select "New Project" to set up your documentation workspace.
      </Step>
    </Steps>
  </Tab>
  <Tab title="API Integration" icon="code">
    Integrate with your existing systems using RESTful APIs and webhooks.
    
    <CodeGroup tabs="JavaScript,Python">
      ````javascript
const response = await fetch('/api/projects', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
const projects = await response.json();
````
      ````python
import requests

response = requests.get('/api/projects', headers={
    'Authorization': 'Bearer YOUR_TOKEN'
})
projects = response.json()
````
    </CodeGroup>
  </Tab>
</Tabs>
